# Next Mind Map App
A simple, lightweight mind mapping app built with Next.js.

## Features
Add, rename, delete nodes. Export mind maps as a JSON. Import previously exported mind maps. Simple UI.

## Tech Stack
* **Framework:** Next.js
* **UI:** React
* **Styling:** Tailwind CSS
* **Icons:** lucide-react
* **Language:** TypeScript

## Getting Started

### Prerequisites
Make sure you have the following installed:

* Node.js (v18 or later recommended)
* npm, yarn, or pnpm

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/daveydhruti/next-mindmap-app.git
cd next-mindmap-app
npm install
```

### Running the App

Start the development server:
```bash 
npm run dev
```

Open this in your browser:
```
http://localhost:3000
```

### Building for Production
```bash
pnpm run build
npm start
```

## Usage
* Use the UI to add, delete, or rename nodes in your mind map
* Export your mind map to save it locally
* Import a previously exported file to restore a mind map

## Scripts
Available npm scripts:
* `npm run dev` – Start the development server
* `npm run build` – Build the app for production
* `npm start` – Run the production build
* `npm run lint` – Run ESLint
